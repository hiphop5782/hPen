hPen
 - 강의용 화면 필기 프로그램입니다 
 
 <사용 방법>
1. 프로그램의 압축을 해제합니다.
2. 프로그램을 실행합니다(트레이에 강아지 모양 아이콘이 생깁니다)
3. 트레이를 우클릭하거나 단축키를 눌러 기능을 실행할 수 있습니다.
4. 원할 경우 바탕화면에 바로가기를 만들거나 시작 프로그램으로 등록할 수 있습니다.
	(윈도우 7 이상만 지원하는 기능입니다)

<기능 소개>

(1) 화면 캡쳐 (단축키 Alt+1)
 - 화면을 스크린샷 캡쳐하는 기능입니다.
 - 픽픽(pickpick)과 유사한 기능을 가지나 아직은 정밀도가 약간 부족합니다.
 - 정밀도는 개선 예정입니다.
 - 기본적으로 버퍼에 복사되어 붙여넣기가 가능하며, 환경설정 변경시 파일로 저장도 가능합니다.

(2) 화면 필기 (단축키 Alt+2)
 - 강의중인 화면에 필기하는 기능입니다.
 - 다중 모니터의 경우에도 지원됩니다.
 - 한글 입력이 가능합니다.(한영 변환은 키 입력 지원 관계로 한영키가 아닌 다른 키로 정하였습니다)
 - 마지막에 저장한 그림을 복원하는 기능이 제공됩니다.
 - 마우스 오른쪽을 클릭하시면 아이콘 스탬프 기능을 사용할 수 있습니다.

[단축키 리스트]
Shift+Space		한/영 전환
Ctrl			한/영 전환
Q			사각형 그리기
W			원 그리기
E			직선 그리기
R			화살표 그리기(역방향)
F			화살표 그리기(정방향)
T			글자 입력 모드
ESC			글자 모드 취소 또는 필기 종료
F1~F10			색상 바꾸기
C			필기 전체 지우기
Ctrl+R			전체 삭제 하였거나 다시 필기하려는 경우 이전 내용을 복원
Ctrl+Z			undo, 뒤로가기
Ctrl+S			필기 내용 저장하기
 
(3) 돋보기 (단축키 Alt+3)
 - 창이 하나 나오며 마우스 커서 주변이 화면에 표시됩니다.
 - 창을 선택한 뒤 창 위에서 마우스 휠을 조작할 경우 배율을 조절할 수 있습니다.
 - 돋보기 창의 크기 변경이 가능하며, 다음 실행시 해당 크기로 자동 표시됩니다.
 - 1배율 부터 10배율 까지 지원합니다.

(4) Windows 돋보기(단축키 Alt+4)
 - 윈도우의 기본 돋보기 기능으로 화면 확대가 가능합니다
 - 종료를 원하실 경우 Alt+4를 다시 한번 누르시면 가능합니다
